# MCP Sentinel — Research Intelligence System
## Continuous 8-Track Research — v1.0

### Track 1: Threat Landscape
- Monitor CVE databases for MCP-related vulnerabilities
- Track security research publications on agentic AI
- Map attack techniques to detection rules
- Cadence: Weekly

### Track 2: Ecosystem Growth
- Monitor registry source health and server counts
- Track new entrants and category evolution
- Update deduplication estimates
- Cadence: Weekly

### Track 3: Competitive Movements
- Track 10 competitors + adjacent platform players
- Monitor product launches, funding, hires
- Update feature gap matrix
- Cadence: Bi-weekly

### Track 4: Regulatory Environment
- EU AI Act enforcement timeline (August 2026)
- NIST AI RMF updates
- MCP Working Group standards
- Cadence: Monthly

### Track 5: Technology Evolution
- MCP protocol updates and new capabilities
- New transport mechanisms (Streamable HTTP)
- Client-side security features
- Cadence: Bi-weekly

### Track 6: Detection Science
- False positive/negative analysis from scans
- New detection techniques from academic research
- Rule performance metrics
- Cadence: Weekly (tied to scanning)

### Track 7: Market Signals
- Enterprise adoption patterns
- Developer community sentiment
- MCP usage in production environments
- Cadence: Monthly

### Track 8: Partnership Opportunities
- Registry operator changes
- Gateway vendor product updates
- MCP Working Group membership changes
- Cadence: Monthly
